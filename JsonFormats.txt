Update Password json format: 
{
  "data": {
    "oldPass": "test@123",
    "newPass": "admin123"
  }
}

Update Bio json format:
{
  "data": {
    "Bio": "Hello I updated my bio"
  }
}

Login json format:
{
  "data" : {
    "email" : "test@gmail.com",
    "password" : "test@123"
  }
}

Register json format:
{
  "data" : {
    "user_id" : "adam.paul",
    "first_name" : "adam",
    "last_name" : "paul",
    "email" : "test@gmail.com",
    "password" : "test@123",
    "bio" : "Hello"
  }
}
